**AI Study Planner** is a Python Streamlit app that predicts your exam performance based on daily habits and provides a personalized study plan. It uses Machine Learning models (regression and classification) to estimate your expected score, risk level, and suggests actionable improvements.

---

## Features

- Predict your **exam score** using daily routines and mock test scores.
- Evaluate your **risk level** (High / Medium / Low).
- Generate a **personalized study plan** based on predictions.
- Interactive **dashboard with charts** to visualize your habits.
- Stylish **gradient background and cards** for a modern look.
